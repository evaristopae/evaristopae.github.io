O que é o TV_7_Guias?

Projeto pessoal não-comercial.

A motivação para criar este projeto foi o de listar de forma simples e filtrada, os filmes transmitidos na TV nacional por cabo nos últimos sete dias, associado a um serviço de distribuição RSS (Wikipedia:RSS).

As listas dos filmes são atualizadas diariamente e estão contemplados apenas os canais sem qualquer subscrição adicional e que, por norma, transmitem longas-metragens.

Os <12> canais abrangidos:

  • RTP Memória
  • Hollywood
  • Cinemundo
  • Star Movies
  • Star Channel
  • AXN
  • Star Life
  • Star Comedy
  • AXN White
  • AXN Movies
  • SyFy
  • AMC

Ressalvas

Este projeto não pretende ser, ou fazer, um arquivo das transmissões. Apenas são listados os filmes dos últimos sete dias, período este que reflete o prazo máximo de recuo de visualização de conteúdos na operadora Vodafone.

Este projeto surge do desejo de simplificar o ato de pesquisa de filmes para possível visualização, sem alguns dos constrangimentos encontrados nos "cardápios" disponibilizados pelas operadoras do serviço. Não se pretende, de forma alguma, alterar o comportamento dos telespectadores na sua pesquisa de conteúdos, mas sim, tornar acessível mais uma opção para a realização desse tipo de consulta.

Este projeto não está isento de erros e a recolha poderá não contemplar todos os canais nem todos os filmes transmitidos.
Por tal, as minhas desculpas.
